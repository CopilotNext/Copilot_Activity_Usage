migrate from csv to mysql

Here are the new features in the latest 20231201 version:

  • for the database, extend support to mysql
  • support .env format parameters
  • design update

for the migrate:

  • it will migrate exising configed organizations which are not already set in mysql
  • to migrate, this applicaiton should be set to use mysql in eithor .env or environment parameters.
  • Only organizations settings, previous fetch data are migrated from csv to mysql
  • the latest activity are not overrides by migrate, as we are not sure whether data in csv are most updated
  • if you want to migrate the latest activity, please use refresh, which will reconsile the migrate data with latest data
